# Empire's Hand

A card game inspired by Age of Empires. Real-time strategy meets deckbuilding, playable on phone, tablet, and desktop.

## Elevator Pitch

What if you could play Age of Empires in card game format, on your phone/tablet/computer?

## Status

🚧 Early development — core mechanics and architecture in progress.

## Tech Stack (TBD)

- Frontend: React + TypeScript
- State: Zustand
- Multiplayer: Socket.io
- Backend: Node.js + Express

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev
```
